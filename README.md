# UCOSIII-LWIP-ENC28J60-STM32f103ZE-WEB
该项目是基于UCOSIII，LWIP，STM32F103ZE，ENC28J60，WEB做的移植。通过web显示板子的温度和时间，并实现web控制LED0~LED5的功能。此外，还带有ucosiii特有功能，如互斥信号量，信号量，事件标志组，消息队列等，这就大概就是控制usart，key，tftlcd，led，lwip等同步通信的功能。
